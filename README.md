## [봄날의 햇살](http://i7a606.q.ssafy.io/#/)
> 자폐 아동 치료 플랫폼 <br>
자폐 아동들의 증상 데이터를 분석해 치료사를 매칭하고, 화상 상담을 통해 치료사와 연결한다. <br>
자폐 증상 치료를 위한 놀이 프로그램을 제공한다.
